## About
`stock_analysis` is a tool to automate daily tasks for stock analysis.

## Usage Requirements
The proect uses [uv](https://docs.astral.sh/uv/) as a package manager. Follow the [installation guide](https://docs.astral.sh/uv/getting-started/installation/) to set it up (if you don't have it yet).

Once `uv` is installed, run:
```
uv run main.py
```
